# Tiendanime - Carrito de Compras

Este proyecto es una tienda online ficticia llamada "tiendanime" donde los usuarios pueden ver productos de anime y agregarlos a un carrito de compras. El objetivo es practicar conceptos de HTML, CSS y JavaScript, incluyendo manipulación del DOM y almacenamiento de datos en el navegador.

## Estructura del proyecto

- `index.html`: Página principal de la tienda.
- `css/estilos.css`: Estilos personalizados para la tienda.
- `js/javascript-primera-vez.js`: Lógica de la tienda y el carrito de compras.
- `imagenes/tenor.gif`: Imagen animada utilizada en el encabezado.

## Funcionalidades principales

- Visualización de productos de anime.
- Agregar productos al carrito.
- Visualización del total de ítems y monto a pagar.
- Botón para simular el pago.
- Notificaciones visuales usando SweetAlert2.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- [SweetAlert2](https://sweetalert2.github.io/)

## Cómo ejecutar el proyecto

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador web favorito.
3. Interactúa con la tienda agregando productos al carrito y simulando el pago.

## Créditos

Proyecto realizado con fines educativos para practicar desarrollo web frontend.
